# Inbox - Входящи заявки

Сложи файлове тук за обработка от AI агента.

## Поддържани формати

### Без Python (винаги работи):
- `.txt` — Plain text
- `.md` — Markdown

### С Python (ако е инсталиран):
- `.eml` — Email файлове
- `.msg` — Outlook съобщения
- `.docx` — Word документи
- `.xlsx` — Excel таблици
- `.pdf` — PDF документи
- `.rtf` — Rich Text Format

## Как работи

1. Сложи файл(ове) тук
2. При старт на сесия агентът проверява inbox/
3. Ако намери файлове — докладва и чака одобрение
4. След одобрение — обработва файловете и създава структурирани заявки

# documents/archive/

Архив на исторически документи от ранните версии на AI_Svetlio.

Тези файлове са запазени за референция, но **не са актуални** и **не се използват** от текущата версия.

| Файл | Описание | Оригинална версия |
|------|----------|-------------------|
| `Svet AI Iron Rules (Железните Прави.txt` | Първа версия на Iron Rules (преди ребрандиране) | v1.0 |
| `SVET_AI_FULL_DOCUMENTATION.md` | Пълна документация на разработката | v1.0.0 |
| `SVET_ECOSYSTEM_ROADMAP2.md` | Визия за бъдещата екосистема | Draft v1.0 |
| `TIMELINE_AND_GAP_ANALYSIS.md` | CargoFlow хронология и gap анализ | CargoFlow |
| `CLIENTREQUESTS_README.md` | CargoFlow ClientRequests документация | CargoFlow v1.0 |
| `REGISTRY.md` | Стар шаблон за регистър (заменен от templates/requests/) | v1.0 |
| `TEMPLATE.md` | Стар шаблон за заявки (заменен от templates/requests/) | v1.0 |

За актуални документи виж:
- `documents/IRON_RULES.md` — текущи Iron Rules
- `documents/USER_GUIDE.md` — наръчник за потребителя
- `documents/REFRESH_REMINDER.md` — напомняне за Context Refresh
